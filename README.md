## What I Do

I specialise in designing and executing **data migrations** across systems, ensuring data integrity, zero loss, and smooth go-lives. My work spans the full lifecycle: source analysis, transformation logic, validation frameworks, and stakeholder reporting.

- **Migration Planning** — requirements gathering, source-to-target mapping, risk assessment
- **ETL Development** — building robust pipelines that handle messy, real-world data
- **Data Validation** — row counts, reconciliation reports, and post-migration audits
- **Integration** — connecting systems via REST APIs and middleware
- **Reporting** — translating migration metrics into clear Power BI dashboards
